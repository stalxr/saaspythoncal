# Python Tasks Project

Проект с реализацией 9 различных задач на Python.

## Структура проекта

```
project/
├── task1_calculator/       - Калькулятор
├── task2_triangle_area/    - Площадь треугольника
├── task3_cylinder_volume/  - Объём цилиндра
├── task4_text_quest/       - Текстовый квест
├── task5_todo_list/        - Список задач
├── task6_budget_calculator/ - Калькулятор бюджета
├── task7_system_info/      - Информация о системе
├── task8_system_monitor/   - Системный монитор
├── task9_weather_parser/   - Парсер погоды
└── main.py                 - Главный файл запуска
```

## Запуск

```bash
python main.py
```

## Требования

- Python 3.8+
- Стандартная библиотека (для задач 1-7)
- psutil (для задачи 8)
- requests (для задачи 9)

Установка зависимостей:
```bash
pip install -r requirements.txt
```

## Задачи

1. **Калькулятор** - базовые арифметические операции
2. **Площадь треугольника** - расчёт площади прямоугольного треугольника
3. **Объём цилиндра** - расчёт объёма цилиндра с заданными параметрами
4. **Текстовый квест** - интерактивная текстовая игра
5. **Список задач** - простой менеджер задач
6. **Калькулятор бюджета** - учёт доходов и расходов
7. **Информация о системе** - сбор данных о компьютере
8. **Системный монитор** - мониторинг CPU, памяти, дисков и процессов в реальном времени
9. **Парсер погоды** - получение данных о погоде через API wttr.in
