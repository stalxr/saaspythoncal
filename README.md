## Задачи

1. **Калькулятор** - базовые арифметические операции
2. **Площадь треугольника** - расчёт площади прямоугольного треугольника
3. **Объём цилиндра** - расчёт объёма цилиндра с заданными параметрами
4. **Текстовый квест** - интерактивная текстовая игра
5. **Список задач** - простой менеджер задач
6. **Калькулятор бюджета** - учёт доходов и расходов
7. **Информация о системе** - сбор данных о компьютере
